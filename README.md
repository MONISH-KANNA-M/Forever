# 👕 Forever – Online Clothing E-Commerce Website

## 📌 Overview
Forever is an online clothing e-commerce web application that allows users to browse, select, and purchase fashion products بسهولة. The platform provides a smooth shopping experience with product listings, cart management, and order processing.

---

## 🚀 Features
- 👤 User registration and login  
- 🛍️ Browse clothing products  
- 🔍 Search and filter items  
- 🛒 Add to cart and checkout  
- 💳 Order placement  
- 📦 Order history tracking  
- 🛠️ Admin panel for product management  

---

## 🧠 Technologies Used
- Frontend: HTML, CSS, JavaScript  
- Backend: Python (Flask) / Node.js  
- Database: MySQL / MongoDB
